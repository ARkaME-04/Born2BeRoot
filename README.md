# Born2BeRoot

## Description
Born2beRoot is a foundational 42 School project designed to introduce students to the fundamentals of system administration, virtualization, and cybersecurity. The project requires setting up a secure, non-graphical Linux server from scratch, often on Debian or CentOS/Rocky Linux, and applying strict security policies.
